【隐藏临时坦克】左边坦克列表上方 -> 坦克筛选 -> 特殊分类 -> 租用坦克 -> 隐藏
【移除过期临时坦克】仓库 -> 第四个,车库里的坦克 -> 租用 -> 点过期的就能卖
【价位】40金=1块,按这个评价礼包价格
弱势并不可怕,最可怕的 是被人淘汰
【T型车体弱点】在首上的上半区域或履带上方有一块延展区域的车体,T字两端的下侧同样是底盘的一部分,非常脆弱,主要是对面骑坡露侧面打
【无头TD】TD共性是车体扩圈糟糕,旋转车体会需要大量时间缩圈,因此++骑坡伸缩,++架炮等对面出来

HEAT为85°跳弹,和HE一样均无转正无口径碾压无穿深距离衰减
【关于优惠券】活跃度太高就会被发垃圾优惠券 ; 想要全场8.5折(包括限时卖的车)这种就十几天不玩再回来打几局就行

Factorio

记品质

  • 最关键的一点是液体没有品质,因此含有 : 物体+液体->单个物体 的配方适合升品质,并且++高产能、多插槽建筑
  • 【注意】速度插件会减品质,所以品质产线没法上速度
  • 需要做传奇的基础材料 : 铁、铜、钢、塑料(其实就是传说煤)、石头 ; 传奇煤可用低温工厂(8个槽,多上产能)变传奇塑料,传奇塑料浇筑出传说轻质框架,传说轻质框架回收出传说铜+传说钢 ; 传说铁就用铁矿就行 ; 传说石头在火山用方解石+岩浆->石头+熔融铜来搞,然后把铜板丢回岩浆 -> 【所以建议火山星开传说】因为多液体配方+岩浆丢东西高效丢垃圾
  • 【推荐思路】平行回转带,每往下一层就是更高品质 ; 矿机也支持品质插件,可以大矿机硬挖
  • 【特别注意】百科里物品是有回收配方的,这个是放入回收机到底出来什么,看清楚了再规划产线(260723搞了个煤+塑料回收品质产线,结果塑料不能回收成煤,白干了)
  • 【品质物品tip】矿机品质只减资源消耗,不加挖矿速度 ; 传送带相关和管道的品质都没用(除了造科研瓶)
