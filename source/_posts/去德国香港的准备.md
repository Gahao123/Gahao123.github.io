## 去VFS站点：大连北站--沈阳站
VFS离沈阳站很近，到沈阳站比较方便，从东广场出。
回来可以原路，沈阳到大连北有比较晚的车。
从大连北站出来回开发区校区：南广场出来，走地铁站地下通道到大路对面再打车，一定不要坐主动找的打表，公交或网约均可

## 办港澳通行证预约
wx小程序移民局12367
物理前往: 高铁直达 ; 飞深圳后高铁香港,深圳到香港只需20m高铁
买境外流量包,去运营商搜"国际",购买后进香港之前去设置 -> SIM卡 -> 开启漫游 ; 还有联通去搜索"跨境语音和短信"打开国际短信
汇丰和众安都可以下APP线上办(在香港下载) , 26年1月后的汇丰one三个月内得有最少10000港币存款,否则收一个月100港币管理费,且人民币换美元最贵,且老z说汇丰是非常听d话的银行 ; 众安za bank,是数字银行,没有线下实体网点一切都在APP进行,入金走跨境支付通很简单,但出金只能用wise汇款,实测往内地转1w转到zfb手续费是160转到银联260 ; 中银现在很难办,基本不可行,拿不出稳定的工资流水(还得6个月带公章)基本要被一刀切,资料还得全纸质打印 ; 【*注*】汇丰和恒生这种最好不要一起开,一张卡风控了会影响另外一张,这些银行都一起的
内卡往港卡免手续费汇:内卡APP首页 -> 转账汇款 -> 跨境汇款 -> 下方"跨境支付通" -> 收款账号+姓名去港卡对应APP找【注,这个只能内卡和港卡同名才能免手续费,且单笔最多1w一天基本上最多3笔】,要转大额就走普通电汇,手续费转越多越划算
众安是自带券商投资功能的,不过得到香港才能办下来
【美股】也是周一到周五,节假日休息 ; 盘中是美东9:30-16:00是主力,对应北京时间夏令时3月-11月是21:30-04:00,冬令时11月-3月是22:30-05:00 ; 盘前盘后夜盘也有但不是主力
汇丰会关户:关户之后汇丰会给一张支票,去香港能把钱提出来,但户头没有且被纳入黑名单 -> 原因:快进快出,应至少存一星期;银行里做一点理财或存款;长期不用,会被认定僵尸户清理

租用地址在美国是很常见的生意。美国人移居海外时也面临同样的问题：在本土没有居所、又没有亲友帮忙，就收不到任何实体邮件。于是很多人把自己的地址拿出来做代收：每天把收到的信件扫描成电子版发给客户，需要时再通过国际快递转寄原件。这类公司很多，随手一搜一大把，比如iPostal1、Anytime Mailbox，月费普遍在10–30美元。
